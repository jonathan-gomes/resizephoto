## ResizePhoto rest api

Consume and resize photos, listing the resized photos on a rest api

### Enviroment requirements
* MongoDB server installed

### Installations
* npm install

### Run

* npm run start

### Test

* npm run test

## Basic usage

## 1 - Update photos from remote server:

* /updatephotos

## 2 - List resized photos:

* /listphotos
